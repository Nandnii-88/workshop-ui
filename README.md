# workshop-ui
# 🚀 Workshop Booking UI

A modern and responsive **Workshop Booking Web Application UI** built using React and Vite. This project provides a clean and interactive interface for users to explore and book workshops easily.

---

## ✨ Features

* 🎨 Modern UI Design (Gradient + Glassmorphism)
* ⚛️ Built with React (Vite)
* 📱 Fully Responsive Layout
* ⚡ Fast Performance
* 🎯 Clean and Structured Code
